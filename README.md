# PlantPlan

Создать веб-приложение на Python, в котором я буду отмечать фазы роста и развития растений, подкормки,
заметки, календарь, по аналогии с программой "Grow with Jane" для Android. 
Растения размещаются в разных локациях, имя которых задаёт пользователь. У каждого пользователя свои локации и растения.
У каждого растения есть таймлайн, где отмечены события и фазы роста в днях/неделях.
проработать БД - postgresql, таблицы, связи и т.д.
